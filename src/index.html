    Tips

    Make sure the various colours in the SVG are non-overlapping paths.

    In Inkscape:

    • Convert all objects into paths (Paths > Objects to Path)
    • Convert any strokes into paths (Paths > Stroke to path)
    • Duplicate any overlapping colour and subtract it from the covered colour (Edit > Duplicate; Path > Difference)

    If you want to have a solid coloured base:

    • Select the colour that bounds your object
    • Duplicate it
    • Path > Break apart
    • Delete all paths inside the bounding path
    • Make it a different colour from all others
    • Set depth to a negative number

    To preserve detail use 'Arachne' slicing.
